# About This Site

This is a Jekyll site built with the **Simplex** theme from [andreondra/jekyll-theme-simplex](https://github.com/andreondra/jekyll-theme-simplex). The site serves as a proof of concept for setting up Jekyll with GitHub Pages.

## Theme Information

We're using the [jekyll-theme-simplex](https://github.com/andreondra/jekyll-theme-simplex) theme, which provides:

- A clean, modern design with light/dark mode support
- Mobile-responsive layout
- GitHub Pages compatibility
- Built-in lightbox for images
- Category and author support
- SEO optimization

## Technology Stack

- **Jekyll**: Static site generator
- **GitHub Pages**: Hosting platform
- **Simplex Theme**: Modern, responsive design with dark mode
- **Markdown**: Content writing format

## Features

✅ **Responsive Design** - Works perfectly on all devices  
✅ **Dark Mode Support** - Automatic light/dark theme switching  
✅ **Category System** - Organize posts by categories  
✅ **Author Profiles** - Support for multiple authors  
✅ **Image Lightbox** - Beautiful image viewing experience  
✅ **SEO Optimized** - Built-in SEO tags and sitemap
